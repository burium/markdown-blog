Мы занимаемся по учебнику Schritte International. Почитать об этом УМК (Учебно- Методический Комплект) можно здесь.<http://lerne-deutsch.ru/schritte-international/>
Давайте разберём, как организован процесс обучения по   УМК (Учебно- Методический Комплект)  Schritte International на примере первой лекции «Guten Tag . Mein Name ist…»

### 2. Hören Sie/ Послушайте

|  |  |
| --- | --- |
| Каждая лекция начинается с занимательной фото-истории. Её эпизоды опираются на лексический и грамматический минимум по теме. То о чём говорят в эпизодах, разбирается на уроках: обсуждаются новые слова, выделяются разговорные выражения, объясняются новые грамматические структуры. С использованием изученного составляются диалоги. *Дома как минимум 1 раз в день ( желательно 2 раза утром и вечером) выполняйте упражнения на прослушивание!* |  |

[su\_highlight background=»#DDFF99″ color=»#000000″ class=»»]Одних уроков не достаточно для быстрого продвижения вперёд. Поэтому я собрала на этом сайте материалы,  которые годятся в качестве домашних заданий.[/su\_highlight]

### 3.  Заучивание . Приложение Quizlet​

|  |  |
| --- | --- |
| Существует хороший способ учить и тренировать новые слова по теме -приложение **Quizlet** (для всех устройств ). Приложение интегрировано на сайте или его можно скачать для мобильных устройств.  ***Альтернатива**: выписывать слова и учить, как в школе (желательно маркером сортировать глаголы, сущ, прил, устойчивые выражения).* | [Ознакомиться с Quizlet](https://quizlet.com/) |

### 4. Тренировка — Интерактивные упражнения к учебнику

|  |  |  |  |
| --- | --- | --- | --- |
| Грамматические и лексические структуры тренируются с помощью интерактивных упражнений по теме от издателя учебника и приложения Deutschtrainer от Гёте- института ( к сожалению только для мобильных устройств). Эти же упражнения используют при подготовке к экзамену. Они имеют возможность самоконтроля, то есть предназначены для самостоятельной работы дома. ***Альтернатива:** упражнения из рабочей тетради, которая находится во второй части учебника. Минус: Неудобно по сравнению с интерактивной формой, требует проверки.* | ​  |  | | --- | |  | |  | |

[su\_highlight background=»#DDFF99″ color=»#000000″ class=»»]Этот сайт создан с целью облегчить нам поиски материала и упростить процесс обучения.  Определите самостоятельно поочерёдность выполнения заданий и темп работы.
В идеале за время прохождения лекции , необходимо:

1. *выполнить все упражнения на прослушивание , включая видео по нескольку раз*
2. *знать все слова c артиклями из Quizlet*
3. *выполнить все интерактивные упражнения на сайте и задания из приложения Deutschtrainer Göethe Institut*
4. *уметь не только отвечать, но и задавать вопросы по теме с помощью визуальной опоры «Поддержание беседы»*.[/su\_highlight]
