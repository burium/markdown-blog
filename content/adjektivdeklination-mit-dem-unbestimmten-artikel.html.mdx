## Was man über die Adjektivdeklination wissen sollte.

Steht das Adjektiv vor dem Nomen, ist das Adjektiv Teil einer Nomengruppe und muss dekliniert werden. Das Adjektiv hat dann eine **attributive Funktion**. **[Adjektive als Nomenattribute](lernen.php?menu_id=18#nomen "Nomenattribut")** charakterisieren oder definieren ein **[Nomen](lernen.php?menu_id=9 "Nomen")**.

Eine Nomengruppe besteht demnach aus:

|  |
| --- |
| **Artikel**    +    **Adjektiv**   +     **Nomen** |

Wie das Adjektiv dekliniert wird, ist von 4 Faktoren abhängig:

* **[Form des Artikels](lernen.php?menu_id=10#nomenbegleiter "Artikelformen")** (**[bestimmt](lernen.php?menu_id=59 "Bestimmter Artikel")**, **[unbestimmt](lernen.php?menu_id=60 "Unbestimmter Artikel")**, **[Nullartikel](lernen.php?menu_id=58 "Nullartikel")**)
  + **Der** **junge** Mann kauft **eine** **rote** Rose.
  + Seine Freundin liebt **rote** Rosen.
* **[Numerus](lernen.php?menu_id=6#konjugation "Numerus")** (**Singular /Plural**)
  + **Das** **kleine** Kind spielt mit seinem Ball.
  + **Die** **kleinen** Kind**er** spielen mit ihren Bällen.
* **[Genus](lernen.php?menu_id=9 "Genus")** (**[maskulin, feminin, neutral](lernen.php?menu_id=52 "Geschlecht")**)
  + **Ein** **fleißiger** Junge macht seine Hausaufgaben.
  + **Ein** **liebes** Mädchen spielt mit seiner Puppe.
  + **Eine** **junge** Frau möchte heute Abend in die Disko gehen.
* **[Kasus](lernen.php?menu_id=16 "Kasus")** (**[Nominativ](lernen.php?menu_id=38 "Nominativ")**, **[Akkusativ](lernen.php?menu_id=39 "Akkusativ")**, **[Dativ](lernen.php?menu_id=40 "Dativ")**, **[Genitiv](lernen.php?menu_id=56 "Genitiv")**)
  + **Ein** **ehrgeiziger** Fußballspieler trainiert täglich mehrere Stunden.
  + Der Trainer beobachtet **einen** **ehrgeizigen** Fußballspieler.
  + Ein Manager bietet **einem** **ehrgeizigen** Fußballspieler einen Profivertrag an.
  + Die Freundin **eines** **ehrgeizigen** Fußballspielers freut sich mit ihm.
