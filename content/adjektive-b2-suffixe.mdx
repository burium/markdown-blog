# Adjektive Список B2. Словообразование. Суффиксы и префиксы

Ihr wollt euren Wortschatz im Deutschen gezielt erweitern und typische Adjektiv-Endungen erkennen? Dann habe ich heute etwas Besonderes für euch: **92 wichtige Adjektive auf B2-Niveau**, gruppiert nach ihren typischen Suffixen (**-bar, -haft, -ig, -lich, -sam, -voll** usw.).

## Составные прилагательные (Zusammengesetzte Adjektive)

Немецкий язык позволяет создавать новые прилагательные, комбинируя разные части речи:

1. **Прилагательное + Прилагательное**  
   Пример: `hell + blau → hellblau` (светло-синий)
2. **Глагол + Прилагательное**  
   Пример: `wissen + begierig → wissbegierig` (любознательный)
3. **Существительное + Прилагательное**  
   Пример: `die Leistung + fähig → leistungsfähig` (производительный)

## Производные прилагательные (Abgeleitete Adjektive)

Образование прилагательных с помощью суффиксов:

### Основные суффиксы и их значения:

1. **-lich, -ig** — описывают качество:

   * `der Stein → steinig` (каменистый)
   * `bedrohen → bedrohlich` (угрожающий)
2. **-isch** — указывает на принадлежность или характеристику:

   * `Franzose → französisch` (французский)
   * `angeben → angeberisch` (хвастливый)
3. **-bar, -abel** — выражают возможность:

   * `ableiten → ableitbar` (выводимый)
   * `variieren → variabel` (изменяемый)  
     *(Суффикс -abel характерен для заимствованных слов)*
4. **-gemäß, -mäßig** — описывают абстрактные качества: (Различия между этими суффиксами [здесь](https://t.me/deutschstube/236))

   * `die Zeit → zeitgemäß` (современный)
   * `das Recht → rechtmäßig` (законный)
5. **-al, -ell, -iell** — используются с заимствованными словами:

   * `die Region → regional` (региональный)
   * `die Potenz → potenziell` (потенциальный)
6. **-iv, -ös** — также для заимствований:

   * `der Impuls → impulsiv` (импульсивный)
   * `der Nerv → nervös` (нервный)

## Отрицание с помощью префиксов

1. **miss-** — отрицание с оттенком ошибки:  
   `verständlich → missverständlich` (непонятный, двусмысленный)
2. **un-** — общее отрицание:  
   `höflich → unhöflich` (невежливый)
3. **in-, ir-, il-, im-, non-** — для заимствованных слов:

   * `stabil → instabil` (нестабильный)
   * `rational → irrational` (иррациональный)
   * `legal → illegal` (незаконный)
   * `verbal → nonverbal` (невербальный)
4. **de-, des-** — аннулирование качества:

   * `aktiviert → deaktiviert` (деактивированный)
   * `interessiert → desinteressiert` (незаинтересованный)
5. **-los** — отсутствие качества:  
   `schlafen → schlaflos` (бессонный)

## Усиление значения через сложные слова

1. **Существительное + Прилагательное**:  
   `das Gift + grün → giftgrün` (ядовито-зелёный)
2. **Глагол + Прилагательное**:  
   `betteln + arm → bettelarm` (нищий как попрошайка)
3. **Прилагательное + Прилагательное**:  
   `hoch + aktuell → hochaktuell` (очень актуальный)
